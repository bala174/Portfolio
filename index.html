Weather Forecasting

Project Description:

The Weather Forecasting Application is a web app that allows users to retrieve real-time weather information for their current location and view a one-week weather forecast for any specified area. It uses JavaScript’s fetch API to interact with the OpenWeatherMap API, ensuring accurate and up-to-date weather data. The user-friendly interface makes it easy to check current weather conditions and plan ahead based on the forecast.

Technology Used:

HTML, CSS, JavaScript
